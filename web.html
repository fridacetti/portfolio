Tema 2 - Web

Læring

I tema 2 lærte jeg at arbejde med Figma samt forskellige filformater til web, herunder PNG, JPG, GIF, SVG og WebP. Jeg lærte, hvordan man komprimerer billeder og ændrer størrelse og format efter behov.

Under dette tema blev jeg også introduceret til designkonventioner, gestaltprincipper, wireframes og styletiles. Jeg lærte om fonte, tekstopsætning, mappestruktur, navigation og navnekonventioner.

Inden for kodning lærte jeg at arbejde ud fra "mobile first"-tilgangen. Jeg arbejdede med CSS Grid, Flexbox og media queries gennem en række øvelser og opgaver. Jeg anvendte HTML og CSS og lærte om IT-sikkerhed, herunder, hvordan man opretter et domæne og sikrer det korrekt.

Desuden lærte jeg at bruge værktøjer som VS Code og FileZilla. Jeg lærte at validere HTML og CSS samt at anvende udviklingsværktøjerne i Firefox og Chrome.

Proces

Opgaverne under tema 2 blev udarbejdet på baggrund af oplæg, givet materiale og materiale, som jeg selv skabte. I processen blev der også undervist i ophavsret, rettighedslicenser og metadata med angivelse af rettigheder i digitale billeder.

Gennem min læring udviklede jeg, med udgangspunkt i udleveret tekst, billeder, farver, typografi og wireframes, et responsivt website, som blev afleveret som studiestartsprøve.
